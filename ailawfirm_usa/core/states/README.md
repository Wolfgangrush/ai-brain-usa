# core/states/ — Per-State Modules

v0.1 ships 6 empty state placeholders (CA · TX · NY · FL · IL · DE). v0.2+ fills with state-specific:
- Court hierarchy
- Procedural rules (state CRC · NYCPLR etc.)
- Bar rules (state-specific publicity restrictions)
- Statutes (CCPA · NY SHIELD · TX consumer · etc.)

Per ADR-006: v0.1 is Federal-focused. State encoding is the largest v0.2 work item.
