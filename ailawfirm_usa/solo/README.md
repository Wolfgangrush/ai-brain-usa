# solo/ — Solo Advocate Mode (v0.1 scope)

This subpackage holds modules used only in solo mode (single-advocate practice):
- Personal practice brain (matter memory across the advocate's career)
- Single-user matter tracker
- Client memory (no multi-user permission layer)

v0.1 builds out the skeleton; real depth lands in v0.2 along with statute text.
Shared functionality (statutes, courts, citations, ontology) lives in `core/`.
