# 欢迎，律师 — 您的AI Brain从这里开始

您不需要成为程序员。您需要一台笔记本电脑（Mac或Windows）、用于安装的网络连接，以及30分钟时间。只要会复制粘贴，就能使用。

> **English:** [GETTING_STARTED.md](GETTING_STARTED.md)

---

## 一句话概述

一款将您的笔记本电脑变成小型私人律所助理的免费工具。它能记住您的案件（每个案卷、每个当事人、每份命令、每个听证日期）。它能验证Bluebook法律引证格式。它能告诉您哪个联邦法院有管辖权。它能标记ABA示范规则第7条的风险和CCPA合规问题。它运行在**您自己的电脑上**——无需云端，无需上传，第三方无法读取您的客户数据。MIT许可证。您拥有您存入的一切。

## 为什么美国独立执业律师应该关心

大律所拥有庞大的律师团队来检查引证、管理案件和起草文件。而您只有自己和一台笔记本电脑。这是您的第二大脑。

- **减少遗忘。** 数月前的案件背景即时重现。
- **减少遗漏期限。** 跟踪听证日期、提交截止日期和诉讼时效。
- **保持ABA合规。** 内置防火墙标记宣传风险（规则7）、跨辖区执业问题（规则5.5）和保密问题（规则1.6）。
- **节省费用。** Westlaw/PACER/LexisNexis每年花费数千美元。这个工具$0。
- **数据私密。** 客户案件永远不离开您的笔记本电脑。
- **可扩展。** 当您雇佣第一位律师时，从独立模式切换到律所模式——同样的工具，同样的数据。

## 包含内容

7位专家驻留在您的终端中：

1. **接待员（大脑）** — 理解您的需求并转接给正确的专家。
2. **引证书记员** — 验证Bluebook引证（SCOTUS、F.3d、F.Supp.3d、U.S.C.、C.F.R.）。
3. **法院登记员** — 了解联邦法院：SCOTUS、13个巡回法院、94个地区法院、破产法院、税务法院。
4. **案件管理员** — 管理您的在办案件（当事人、听证、命令、截止日期）。
5. **起草助理** — 连接wolfgang_rush起草插件（v0.2+）。
6. **合规官** — ABA规则7、5.5、1.6、正式意见512（AI伦理）、CCPA/CPRA、FinCEN CTA。
7. **期限追踪器** — 诉讼时效、提交截止日期、听证日期（FRCP规则6时间计算）。

```bash
ailawfirm_usa
```


## 范围说明
