# Requests for Production of Documents and ESI (Rule 34)

## Source URLs
- [Fed. R. Civ. P. 34](https://www.law.cornell.edu/rules/frcp/rule_34)

## Last Verified
2026-05-29

## Authority Tier
Primary — FRCP Rule 34.

## Currency Warning
2015 Rule 34 amendments require objections to specify whether responsive materials are being withheld on the basis of objection. Generic objections "subject to and without waiving objections" insufficient; if not withholding, say so; if withholding, identify scope.

## Template Type
Discovery Request Set.

## Structural Skeleton
1. **Caption** — "PLAINTIFF'S FIRST REQUESTS FOR PRODUCTION OF DOCUMENTS AND ESI TO DEFENDANT."
2. **Definitions and Instructions** — incorporate Local Rule definitions; identify time period; ESI format request (native + metadata, or load-file format); privilege log requirement (Rule 26(b)(5)).
3. **Document Categories** — for each: precise description, scope (custodians, date range, subject matter).
4. **ESI Specifications** — preserve metadata; produce in agreed format (load-file with TIFF/PDF + native for spreadsheets; text + metadata fields); reasonable search terms (per Rule 26(f) protocol).
5. **Time and Place for Production** — date; format (electronic via FTP or production tool; paper copies as required).
6. **Signature Block.**

## Drafting Conventions
- Each request must "describe with reasonable particularity each item or category" (Rule 34(b)(1)(A)).
- Responses due within 30 days (Rule 34(b)(2)(A)); shorter or longer by stipulation/order.
- Rule 34(b)(2)(C): an objection must state whether any responsive materials are withheld on the basis of that objection.
- Coordinate with Rule 26(f) ESI protocol — search terms, custodian list, date range, production format pre-agreed.
- Use Rule 45 subpoenas for non-party productions.
- Rule 34(c) cross-references inspection of premises — pair with separate motion if needed.

## Fair-Use Excerpt
> "An objection to part of a request must specify the part and permit inspection of the rest. The responding party may state that it will produce copies of documents or of electronically stored information instead of permitting inspection. The production must then be completed no later than the time for inspection specified in the request or another reasonable time specified in the response. An objection must state whether any responsive materials are being withheld on the basis of that objection." — Fed. R. Civ. P. 34(b)(2)(C).

## Practical Use for Solo Advocate / Small Firm
Workhorse document-discovery instrument. Coordinate with ESI protocol from Rule 26(f) conference.
