# USA Drafting Corpus — Summary

## What this corpus is

89 federal-first drafting templates covering: regulatory compliance (privacy / cyber / healthcare / AI governance / state AI acts), commercial contracts (MSA / SaaS / employment / independent contractor / NDA / asset and stock purchase / shareholders), federal civil pleadings (complaint / answer / counterclaim / third-party / amended / derivative / class / removal), motions (12(b)(6) / 56 / 65 / 37 / in limine / 12(b)(3) / 29 / 1447 / 4 FAA), discovery (Rule 26 / 33 / 34 / 36 / 30 / privilege log / ESI protocol), court forms (appearance / cover sheet / summons / subpoena / protective order / 26(f) report / 7.1 disclosure), appeals and enforcement (FRAP 3 / appellant brief / 35 / cert / writ of execution / garnishment / charging order), brief skeletons (SJ / Daubert / Winter / engagement letter / litigation hold), ADR (mediation statement / AAA demand / settlement and release / EEOC charge / DMCA takedown), style (Bluebook 21), insolvency (Ch 7 / 11 / 13 / proof of claim / relief from stay), tribunals (Tax Court / Immigration BIA / ALJ / SSA / APA complaint), commercial backbone (purchase order / supply / distribution / franchise / loan / security agreement), corporate (DE certificate / bylaws / resolutions / merger / Series Seed / Form D), and securities-corporate regulatory (10-K / 8-K / § 220 demand / FCPA compliance policy).

## Uniform template shape

Each template uses:
1. **Source URLs** — primary statutes, rules, and authoritative agency / court sources.
2. **Last Verified** — date.
3. **Authority Tier** — primary / secondary / common-law / skeleton.
4. **Currency Warning** — pinned doctrinal currents and 2024-2026 developments.
5. **Template Type** — pleading / motion / contract / brief / notice / form.
6. **Structural Skeleton** — numbered with sub-bullets.
7. **Drafting Conventions** — non-obvious mechanics specific to federal practice.
8. **Fair-Use Excerpt** — verbatim statutory or rule text.
9. **Practical Use for Solo Advocate / Small Firm** — how the template fits into practice flow.

## Doctrinal currency anchored

- **Twombly / Iqbal** plausibility (Rules 8, 12(b)(6), motions, pleadings).
- **Winter** four-factor injunction test (Rule 65, skeletons).
- **Daubert / Kumho / Joiner + 2023 Rule 702 amendment** (motion practice, skeletons).
- **Loper Bright** end of *Chevron* deference; *Corner Post* APA 6-year statute (APA complaints, judicial review).
- **Carpenter / Riley / Van Buren** digital privacy (suppression motions, CFAA templates).
- **Bostock / Muldrow / Groff / Murray v. UBS** employment discrimination (EEOC charge, settlement agreement).
- **Helix v. Hewitt + 2024 DOL FLSA Final Rule litigation** (employment contracts, independent contractor).
- **Spizzirri + Bielski + Henry Schein + Bissonnette** FAA practice (motion to compel arbitration).
- **Hughes v. Northwestern University** ERISA fiduciary (employment).
- **Harrington v. Purdue Pharma** bankruptcy third-party releases (Chapter 11).
- **TransUnion / Lujan / Spokeo** standing (every pleading / APA complaint).
- **Slack / Macquarie** Securities Act § 11 + Rule 10b-5 pure-omissions (securities templates).
- **DGCL 2024 § 122(18) + 2022 § 102(b)(7) officer exculpation + Zuckerberg demand-futility test** (corporate, shareholders, derivative).
- **Tornetta v. Musk + MFW** controller transactions (M&A, shareholders).
- **CFPB v. CFSA + Seila Law** financial regulation (Dodd-Frank reference).
- **2023-2024 SEC final rules** — cybersecurity Item 1.05, clawback § 954 (8-K, FCPA, securities).
- **ABA Formal Opinion 512 (2024)** AI use (counsel engagement letter, drafting style).
- **SPEAK OUT Act + § 162(q)** sexual harassment settlements.

## What this corpus is NOT

- Not state-procedural rules (CA CRC, NY CPLR, TX TRCP, DE Court of Chancery Rules) — deferred to v0.3+.
- Not local rules of individual U.S. District Courts — drafter must consult.
- Not a substitute for primary-source verification at the time of filing.
- Not legal advice. Use under ABA Model Rule 1.1 (competence) and Rule 5.5 (jurisdictional licensure).
- Not pre-litigation strategy (e.g., MDL practice, multidistrict coordination) — separate annex.
- Not sectoral specialist work (FDA premarket; FERC orders; EPA NEPA practice; international trade Section 301; Section 337 ITC) — separate annexes in v0.3+.

## How to use with the firm's drafting agent

The drafting agent loads the relevant template(s) into its context window when matter taxonomy matches the template's domain. Verbatim statutory text and structural skeleton serve as the authoritative reproduction layer for pleadings, motions, contracts. The "Drafting Conventions" section codifies non-obvious mechanics so the agent does not need to learn them empirically.

## Source canon

All templates cite primary U.S. sources: U.S. Code (Office of Law Revision Counsel), Federal Rules (Administrative Office of U.S. Courts), Supreme Court Rules (supremecourt.gov), agency dockets (SEC, FTC, DOJ, DOL, HHS, EEOC), Delaware Code (delcode.delaware.gov), and authoritative secondary (NVCA model documents, LSTA model loan provisions, ULC uniform-act drafts).
